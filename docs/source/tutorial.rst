Usage
=====

The ssc-raft algorithm is called as ``sscRaft`` python module.

.. code-block:: python

    import sscRaft

    '''
    Set input data:

    input1 = ...
    input2 = ...
    
    Set dictionary parameters:

    dic = {}
    ...
    '''

    output = sscRaft.function_name(input1, input2, dic)
