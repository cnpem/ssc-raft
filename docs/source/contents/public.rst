Public
======

***************
Getting started
***************

This page documents the ssc-raft module from Sirius Scientific Computing Group. 

***************
Code structure
***************

This module is implemented in ``CUDA`` with an interface in ``Python 3``
 
****************
Modules
****************

Module call
***************

The ssc-raft algorithm is called as ``sscRaft`` module in ``Python 3`` 

.. code-block:: python
    
   import sscRaft

The functions are called as

.. code-block:: python

   data = sscRaft.function(...)
